## вҡЎ Why sindhinltk?

Sindhi is spoken by **~80 million people** yet has virtually zero open-source NLP tooling. This library changes that.

| | sindhinltk | General NLP libs |
|---|---|---|
| Sindhi Arabic script | вң… Native | вқҢ Broken / mangled |
| Zero dependencies | вң… Pure Python | вқҢ Heavy installs |
| Morpheme-aware stemmer | вң… Sindhi suffix rules | вқҢ Not available |
| Sindhi stopwords | вң… 143 words, 10 categories | вқҢ Not available |
| Sentiment in Sindhi | вң… Labels in ШіЩҶЪҢЩҠ | вқҢ Not available |
| Diacritic handling | вң… Full harakat support | вқҢ Not available |

## рҹ“– Usage

### Tokenizer
```python
from sindhinltk.tokenizer import SindhiTokenizer

tok = SindhiTokenizer()

tok.tokenize("ЪӘШ§ЩҲЪҷЩҠЩҶШҜЪҷ Щ…Ш§Ъ»ЩҮЩҲ ЪҜЪҫШұ Ыҫ Щ…ШіШҰЩ„Ш§ ЩҫЩҠШҜШ§ ЪӘЩҶШҜЩҲ ШўЩҮЩҠ")
# вҶ’ ['ЪӘШ§ЩҲЪҷЩҠЩҶШҜЪҷ', 'Щ…Ш§Ъ»ЩҮЩҲ', 'ЪҜЪҫШұ', 'Ыҫ', 'Щ…ШіШҰЩ„Ш§', 'ЩҫЩҠШҜШ§', 'ЪӘЩҶШҜЩҲ', 'ШўЩҮЩҠ']

tok.sent_tokenize("ЩҮЩҲ ЪҜЪҫШұ Ыҫ ШўЩҮЩҠЫ” ШіЩҶЪҢЩҠ Щ»ЩҲЩ„ЩҠ ШіЩәЩҠ ШўЩҮЩҠЫ”")
# вҶ’ ['ЩҮЩҲ ЪҜЪҫШұ Ыҫ ШўЩҮЩҠ', 'ШіЩҶЪҢЩҠ Щ»ЩҲЩ„ЩҠ ШіЩәЩҠ ШўЩҮЩҠ']
```

### Normalizer
```python
from sindhinltk.normalizer import SindhiNormalizer

norm = SindhiNormalizer()

norm.normalize("ЪҫЩҲЩҸ ЪҫШұ ШұЩҲШІЩҺ Ш§ЩҗШіЪӘЩҲЩ„ ЩҲЩҺЪғЩҠ ЩҝЩҲ", remove_diacritics=True)
# вҶ’ 'ЪҫЩҲ ЪҫШұ ШұЩҲШІ Ш§ШіЪӘЩҲЩ„ ЩҲЪғЩҠ ЩҝЩҲ'
```

### Stemmer
```python
from sindhinltk.stemmer import SindhiStemmer

stm = SindhiStemmer()

stm.stem("ЪӘШ§ЩҲЪҷЩҠЩҶШҜЪҷ")  # вҶ’ 'ЪӘШ§ЩҲЪҷ'
stm.stem("ЩҮЩ„ЩҶШҜЩҠ")      # вҶ’ 'ЩҮЩ„'
stm.stem("ЩҲЪғЩҶШҜШ§")      # вҶ’ 'ЩҲЪғ'

stm.stem_tokens(['ЪӘШ§ЩҲЪҷЩҠЩҶШҜЪҷ', 'ЩҮЩ„ЩҶШҜЩҠ', 'ЩҲЪғЩҶШҜШ§'])
# вҶ’ ['ЪӘШ§ЩҲЪҷ', 'ЩҮЩ„', 'ЩҲЪғ']
```

### Stopwords
```python
from sindhinltk.stopwords import SindhiStopwords

sw = SindhiStopwords()

sw.remove_stopwords(['ШіЩҶЪҢЩҠ', 'Щ»ЩҲЩ„ЩҠ', 'ШўЩҮЩҠ', 'ШӘЩ…Ш§Щ…', 'ШіЩәЩҠ'])
# вҶ’ ['ШіЩҶЪҢЩҠ', 'Щ»ЩҲЩ„ЩҠ', 'ШіЩәЩҠ']

sw.is_stopword("ШўЩҮЩҠ")   # вҶ’ True
sw.get_categories()
# вҶ’ ['pronouns', 'postpositions', 'conjunctions', 'auxiliaries', ...]
```

### Sentiment
```python
from sindhinltk.sentiment import SindhiSentiment

sa = SindhiSentiment()

sa.analyze("ШіЩҶЪҢЩҠ Щ»ЩҲЩ„ЩҠ ШӘЩ…Ш§Щ… ШіЩәЩҠ ШўЩҮЩҠ")  # вҶ’ 'Щ…Ш«ШЁШӘ'   (positive)
sa.analyze("ЩҮЩҲ ШӘЩ…Ш§Щ… Ш®ШұШ§ШЁ Щ…Ш§Ъ»ЩҮЩҲ ШўЩҮЩҠ")   # вҶ’ 'Щ…ЩҶЩҒЩҠ'   (negative)
sa.score("ШіЩҶЪҢЩҠ Щ»ЩҲЩ„ЩҠ ШӘЩ…Ш§Щ… ШіЩәЩҠ ШўЩҮЩҠ")    # вҶ’ 2.0

sa.analyze_detail("ШӘЩ…Ш§Щ… ШіЩәЩҠ Щ»ЩҲЩ„ЩҠ")
# вҶ’ {'label': 'Щ…Ш«ШЁШӘ', 'score': 2.0,
#    'hits': {'positive': ['ШіЩәЩҠ'], 'intensified': ['ШіЩәЩҠ'], ...}}
```

### Full Pipeline
```python
from sindhinltk.tokenizer  import SindhiTokenizer
from sindhinltk.normalizer import SindhiNormalizer
from sindhinltk.stemmer    import SindhiStemmer
from sindhinltk.stopwords  import SindhiStopwords
from sindhinltk.sentiment  import SindhiSentiment

def process(text):
    norm   = SindhiNormalizer()
    tok    = SindhiTokenizer()
    sw     = SindhiStopwords()
    stm    = SindhiStemmer()
    sa     = SindhiSentiment()

    clean  = norm.normalize(text, remove_diacritics=True)
    tokens = tok.tokenize(clean)
    tokens = sw.remove_stopwords(tokens)
    stems  = stm.stem_tokens(tokens)
    label  = sa.analyze(text)

    return {"stems": stems, "sentiment": label, "score": sa.score(text)}

process("ШіЩҶЪҢЩҠ Щ»ЩҲЩ„ЩҠ ШӘЩ…Ш§Щ… ШіЩәЩҠ ЫҪ ЩӮШҜЩҠЩ… ШўЩҮЩҠ")
# вҶ’ {'stems': ['ШіЩҶЪҢЩҠ', 'Щ»ЩҲЩ„ЩҠ', 'ШіЩәЩҠ', 'ЩӮШҜЩҠЩ…'], 'sentiment': 'Щ…Ш«ШЁШӘ', 'score': 2.0}
```

## рҹ§  Linguistic Notes

- **Script**: Sindhi uses the **Naskh Arabic script** with unique letters: `Ъ„ Ъғ Щ» ЪҰ Ъі Ъұ Ъ» ЪҸ ЪҠ Щә ЩҪ ЪҮ ЪҰ`
- **Morphology**: Agglutinative verb system вҖ” a single verb root can generate 40+ surface forms via suffixation
- **Diacritics**: Harakat marks (zabar, zer, pesh) are common in formal/religious text вҖ” the normalizer handles all of them
- **RTL**: Full right-to-left support throughout the library
